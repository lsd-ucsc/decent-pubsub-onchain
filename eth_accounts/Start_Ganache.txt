To start Ganache and make new keys:

ganache-cli -d -a 20 --network-id 1337 --wallet.accountKeysPath [path_to_decent_lib]/decent-pubsub-onchain/eth_accounts/keys.json

Start Ganache with keys.json folder existing:
ganache-cli -d -a 20 --network-id 1337

-d = deterministic (deterministic private keys for testing)

-a 20 = create 20 accounts
